# Projeto 2 (Express + MongoDB)

Este é um projeto de API back-end desenvolvido com Node.js, Express e MongoDB, criado como parte do segundo desafio/back-end da minha formação. A aplicação consiste em um servidor HTTP que expõe endpoints REST para operações básicas de backend, estruturando lógica de rotas, integração com banco de dados e organização de camadas de aplicação.

## Principais Funcionalidades

- Servidor web construído com Express.js

- Conexão e persistência de dados usando MongoDB

- Estrutura de rotas REST para operações de dados

- Organização de código separando responsabilidade entre servidor, rotas e dados

## O que este projeto demonstra

Este repositório evidencia competências importantes para desenvolvimento backend:

- Configuração de servidor HTTP em Node.js

- Projeto e implementação de API REST

- Integração com banco de dados NoSQL (MongoDB)

- Gerenciamento de dependências com npm e organização de projeto

- Uso de padrões de estrutura para aplicações backend

## Como usar pelo terminal

### primeiro instale o projeto
 - git clone https://github.com/tiagomsouzac/PROJETO-2-Back-End.git
 - cd PROJETO-2-Back-End
 - npm install

## Rodar o projeto no diretório do repositorio
 - npm start
